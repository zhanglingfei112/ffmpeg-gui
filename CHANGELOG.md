# Changelog

## [v0.1.0] — 2026-05-31

### ✨ Initial Release

第一个正式版本，基础的视频处理桌面工具。

#### 功能

- 🔄 **格式转换**：MP4 / MKV / WebM / AVI / MOV / GIF 互转
- ✂️ **视频裁剪**：按时间起止或时长截取片段
- 📎 **视频合并**：多个同编码视频合并为一个
- 🎵 **提取音频**：MP3 / WAV / AAC / FLAC / OGG
- 🖼️ **提取帧**：按频率提取为 JPG
- 📉 **压缩视频**：CRF 模式，5 级预设
- 📐 **调整分辨率**：常见分辨率预设
- ⏩ **变速播放**：调整播放速度
- ℹ️ **媒体信息**：详尽的编码信息查看
- 📋 **批量处理**：任务队列 + 并发控制

#### 技术特性

- PySide6 跨平台桌面界面
- 深色主题（Dark Theme）
- 中/英双语国际化支持（启动时自动选择）
- FFmpeg 自动检测与安装引导
- 智能资源调度（并发控制 + CPU 线程限制 + 进程优先级）
- GitHub Actions 自动构建 Windows .exe
